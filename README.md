# Electron React Typescript Boilerplate

Boilerplate code for an Electron and React application using Typescript

## Features

-   Electron
-   Electron Forge
-   React
-   Typescript
-   Eslint
-   Prettier
-   React Developer Tools for Electron
-   Yarn

## Getting Started

To get the boilerplate code only and nothing else

```sh
git clone -b main --single-branch --depth 1 https://github.com/sebsojeda/electron-react-typescript-boilerplate my-app
```

then simply install and run the application

```sh
yarn

yarn start # runs the react application in a regular browser window

# OR

yarn dev # builds and runs the react application with electron
```
